# Proyecto Tienda OLIMPO APIs

> Este proyecto esta basado en nodejs

- CRUD productos

  - Código
  - Nombre
  - Precio
  - Descripción
  - Categoria
  - Marca
  - Talla
  - Colores

- CRUD usuarios

  - identificación (id)
  - nombres
  - apellidos
  - fecha de nacimiento
  - correo
  - clave
  - rol

  - Roles: clientes
  - Roles: empleados
  - Roles: proveedores

## Librerías

- https://expressjs.com/es/ `npm install express --save`
- Nodemon (Hot-reload) `npm install -g nodemon`
- Postgres https://node-postgres.com/ `npm install pg`
